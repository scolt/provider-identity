Yellow Bat Team Identity Provider
identity-as-a-service

This service provide ability to any application to use social media authorization by oauth.

Only one thing that you should do add link to the your identity adapter on your page and then exchange the code for the token

How it works

  1. The user click on the link (e.g. https://api.webtech.by/identity/ivwa/login?redirect_url=https://vk.com)
  2. The user is redirected to the your styled page and have the options for authorization
  3. The user is authorized by any social media and redirected back to your application
  4. Your backend application should read the code from URL and exchange it to JWT token by API
  5. Your application can read the user details by making call to API

API

  • POSThttps://idenity.webtech.by/tokenThe API for exchange code to token ({ code: string})

Supported Clients

  • Facebook
